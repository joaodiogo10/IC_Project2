\doxysubsection{Detailed Description}
Class that codes, to a file, or encodes, from a file, golomb codes. 

This class uses \mbox{\hyperlink{classBitStream}{Bit\+Stream}} in order to read bits from a file or write bits to a file. All errors are handled by \mbox{\hyperlink{classBitStream}{Bit\+Stream}} class. Methods from this class simply return false, propagating an error condition, in case an error occurs in a \mbox{\hyperlink{classBitStream}{Bit\+Stream}} operation. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{classBitStream}{Bit\+Stream}} 
\end{DoxySeeAlso}
