\doxysubsection{Detailed Description}
Class to read and write bits from a file. 

All errors are handled by this class, simply printing to standard output an error or warning massage. Methods that can cause an error to occur return false when it occurs.

{\footnotesize\ttfamily bool Bit\+Stream\+::read\+NBits (\begin{DoxyParamCaption}\item[{unsigned char $\ast$}]{bits,  }\item[{const unsigned int}]{n\+Bits }\end{DoxyParamCaption})}



Read N bits from \mbox{\hyperlink{classBitStream}{Bit\+Stream}}. 

All bytes of the array are written in a little endian fashion. If {\ttfamily n\+Bits} \texorpdfstring{$<$}{<} 8, the reaminder bits are written to the first "{}{\ttfamily n\+Bits} \% 8"{} LSBs in little endian.

Ex\+: ~\newline
 Next bits of file stream = 1010 0000 1010 1111 ~\newline
 n\+Bits = 12.

Result\+: ~\newline
{\ttfamily bits}\mbox{[}0\mbox{]} = 1010 0000 ~\newline
{\ttfamily bits}\mbox{[}1\mbox{]} = 0000 1010 ~\newline
