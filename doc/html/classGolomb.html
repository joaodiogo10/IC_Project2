Detailed Description

Class that codes, to a file, or encodes, from a file, golomb codes.

This class uses BitStream in order to read bits from a file or write bits to a file. All errors are handled by BitStream class. Methods from this class simply return false, propagating an error condition, in case an error occurs in a BitStream operation.
