Detailed Description

Class to read and write bits from a file.

All errors are handled by this class, simply printing to standard output an error or warning massage. Methods that can cause an error to occur return false when it occurs.

◆ readNBits()

bool BitStream::readNBits ( unsigned char *  bits,
const unsigned int  nBits 
)

Read N bits from BitStream.

All bytes of the array are written in a little endian fashion. If nBits < 8, the reaminder bits are written to the first "nBits % 8" LSBs in little endian.

Ex:
Next bits of file stream = 1010 0000 1010 1111
nBits = 12.

Result:
bits[0] = 1010 0000
bits[1] = 0000 1010

Parameters
bitUnsigned char array containing read bits.
nBitsNumber of bits to be read.
Returns
True, if no errors occurred.
False, if errors occurred.
